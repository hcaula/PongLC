# PongLC
An implementation of Pong in Haskell.

## How to run

Make sure you have the full Haskel Platform installed, available [here](https://www.haskell.org/platform/).
And Fungen too. To install it run:

```
sudo cabal update
sudo cabal install fungen
```

After that, run:

````
cd [Project_Path]/src
ghc Main.hs
Main
````
